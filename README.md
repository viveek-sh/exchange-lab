# exchange-lab

A full-stack exchange simulator built as a learning project to explore order books, matching engines, real-time market data, and high-frequency trading system design.

## Tech Stack

- Backend: Node.js, Express, PostgreSQL, Prisma, Redis, WebSockets
- Frontend: Next.js, shadcn/ui
- Monorepo: Turborepo
